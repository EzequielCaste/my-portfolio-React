## Overview

![screenshot](https://i.imgur.com/4y5N9Kr.png)

The demo for the solution to this challenge is hosted on GitHub Pages. I used SASS, Flexbox, CSS Grid and Media Queries to make the page responsive for Mobile, Tablet and Desktop sizes. I start the page design mobile first and later work on the desktop and tablet sizes.

## How To Use

<!-- Example: -->

To clone and run this application, you'll need [Git](https://git-scm.com) and [Node.js](https://nodejs.org/en/download/) (which comes with [npm](http://npmjs.com)) installed on your computer. From your command line:

```bash
# Clone this repository
$ git clone https://github.com/your-user-name/your-project-name

# Install dependencies
$ npm install

# Run the app
$ npm start
```
